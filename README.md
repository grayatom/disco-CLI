_disco-cli v0.0.1_

### To install **disco-cli**, run:

```
$ pip install git+https://github.com/grayatom/disco-CLI.git
```

Disco CLI provide the following commands :

1. whoami
2. list
3. describe
4. status
5. log

### 1. The **whoami** command

-This command prints the email id of the user.

_Example:_

```
$ disco whoami
```
